
Stock Service 
 
     Stock service is microservice which is build on python using fast api 

APIS

  * Create stock, it acccept the stock id, name and quantity, and it get stored in the backend table with stock id and item name , current qty and created date and updated date and created by, updated by

   * update stock, it accepet the stock id name and quantity and it get stored in the backend table with stock id and item name , current qty and created date and updated date and created by, updated by

   * delete the stock, it accept the stock name or id, it delte the data from stock back end table

   * list the stock quantity, it list the item name and stock id and current qty with pagingnation
